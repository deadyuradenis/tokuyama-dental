Условия начисления: при накоплении одним участником акции 60 баллов.
Призы начисляются случайным образом. Период начисления призов: с 01.03.2021 по 31.10.2021. Сроки вручения призов: с 01.03.2021 по 30.04.2022. Начисление призов указывается в личном кабинете участника акции. Для получения приза участник должен указать в личном кабинете контактные данные и адрес доставки.
